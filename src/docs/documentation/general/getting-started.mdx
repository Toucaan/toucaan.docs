# Getting Started

It is very easy to start using Toucaan on your projects.

## Start on a New Project

Paste the following `style` tag in the `<head></head>` section of your application layout to quickly get started:

```html
<style>
  /* Portrait â‡‹ landscape switch. */
  @import url('./public/dist/portrait.css') only screen and (orientation: portrait);
  @import url('./public/dist/landscape.css') only screen and (orientation: landscape);

  /* --bu stands for base unit of blockscoped typography. */
  @media only screen and (orientation: portrait) {
    :root {
      --bu: calc(100vw/100); /* Fallback if vmin is not supported. */
    }
  }
  @media only screen and (orientation: landscape) {
    :root {
      --bu: calc(100vh/100); /* Fallback if vmin is not supported. */
    }
  }
  /* Global variables below */
  :root {
    --bu: 1vmin;
    /* all variables here */
  }
</style>
```



## Start on an Existing Project

Remove all style inclusions like `<link rel="stylesheet" media="all" href="/path/to/bootstrap.css">` from your HTML.
